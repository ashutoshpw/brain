# Reading List — Books, Newsletters & Blogs

A catalog of reading resources worth returning to. The entries here are vetted signal sources — not a firehose. Quality over quantity.

## How to use

- **Books in progress** → tracked in `workspace/areas/learning/HABITS.md` as active habits.
- **Books queued** → add to the `GOALS.md` of the `learning` area.
- **Resources found useful** → add an entry here; bump `updated`.

## Curation principle

Only add a resource if you'd recommend it to your past self without hesitation. Remove resources that have gone quiet or whose signal-to-noise ratio has degraded. This is a living list, not an archive.
