# Roadmap — learning

## Confirmed

1. Seed `workspace/references/books.md` with current reading list — read, reading, and queued books with status and distillation status.
2. Distil first batch of YouTube videos from `watch history` into `workspace/resources/expert-guide/youtube/` via `/youtube` skill.
3. Define 90-day learning curriculum (primary + secondary skill) aligned with career area goals and write it into `workspace/areas/learning/resources/curriculum-2026-Q3.md`.
4. Establish daily reading block habit (see HABITS.md) and track streak for first month.
5. Create `workspace/areas/learning/resources/skill-tracker.md` — a self-assessed proficiency table for all active skills.

## Proposed candidates

- **[proposed]** Spaced-repetition review system: a weekly `resources/reviews/YYYY-Wn.md` that surfaces 5–10 key concepts from the past month's reading for active recall.
- **[proposed]** `workspace/references/courses.md` catalog: structured list of online courses with platform, completion status, key takeaways field.
- **[proposed]** Learning retro: quarterly `resources/retro-YYYY-Qn.md` covering what was learned, what stuck, what was wasted, and what the next quarter's curriculum should be.
- **[proposed]** "Best ideas" digest: a curated `resources/best-ideas.md` pulling the top 3 insights from each book/video distilled — a quick-reference wisdom file.

- **[proposed]** Podcast distillation: extend the `expert-guide` pattern to podcasts — `resources/expert-guide/podcasts/<id>/SUMMARY.md`.
