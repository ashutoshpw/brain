# Principles — learning

## One-liner

Learn deliberately, distil relentlessly, and only count something as "learned" when you can use it without looking it up.

## Values

- **Depth over breadth** — one skill mastered is worth ten half-started. Finish books. Complete courses. Go deep before going wide.
- **Distillation as learning** — consuming is not learning; writing a summary in your own words is. No SUMMARY.md = it didn't count.
- **Compounding** — each new thing learned should connect to something known. Build a latticework, not a pile.
- **Deliberate practice** — passive re-reading builds false confidence. Active recall, practice problems, and teaching others build actual skill.
- **Alignment** — learning should be strategic, not just interesting. Tie the curriculum to career goals and life priorities.

## Decision rules

- If you haven't opened a book in 5 days, it's not a time problem — it's a priority problem. Reschedule or admit you've abandoned it.
- If a concept hasn't been recalled in 14 days, schedule a review before it fades to zero.
- When choosing between two learning resources on the same topic: choose the one with exercises / practice, not the one that's shorter.
- Before starting any new course, write one sentence: "I am taking this course so I can ___." If you can't finish the sentence, don't start.
- If something has been in the learning queue for > 90 days untouched, it's not a priority — archive it or schedule it for next quarter explicitly.

## Anti (what I don't do)

- **Don't mistake highlighting for learning** — if you're not writing it in your own words, you're just collecting.
- **Don't start a second book before distilling the first** — unless the second is explicitly reference material.
- **Don't let the queue become a graveyard** — an ever-growing reading list with no completions is anxiety, not learning.
- **Don't confuse familiarity with mastery** — "I've heard of this" is not the same as "I can use this under pressure."
- **Don't optimise for content speed** — 2× playback speed on a complex talk is not learning; it's checkbox-ticking.
- **Don't learn in isolation** — share what you're learning in conversations, teach it, or apply it. Unused knowledge decays.
