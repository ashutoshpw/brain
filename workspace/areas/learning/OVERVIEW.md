# Overview — learning

## Scope

This area covers deliberate skill acquisition, knowledge building, and information distillation across books, courses, videos, and structured practice. It includes deciding *what* to learn, tracking progress through learning artefacts, and converting consumed information into usable knowledge.

Learning distillation from YouTube videos feeds directly from `workspace/resources/expert-guide/youtube/` — each video summary there is a potential learning unit for this area. Books and courses are tracked in `workspace/references/` catalogs.

## What "good" looks like

- 1–2 books finished per month, with a distillation note in `workspace/references/books.md`.
- Every meaningful YouTube video distilled into `workspace/resources/expert-guide/youtube/<id>/SUMMARY.md` via `/youtube`.
- A clear "learning queue" (what's next) aligned with career and health goals — not just random curiosity.
- Spaced-repetition reviews built into weekly habit (review last week's key concepts on Fridays).
- Skills being built are measurable: not "learning TypeScript" but "can build a validated CLI tool without reference docs".
