# Habits — learning

## Daily

- [ ] Read for at least 20 minutes (physical book, ebook, or long-form article — not social feed)
- [ ] Log the pages / chapter read in today's `focus-YYYY-MM-DD.md` (even a single line keeps the streak honest)
- [ ] When a key idea surfaces during reading, capture it immediately in a scratch note (don't rely on memory)

## Weekly

- [ ] Friday spaced-repetition review: spend 30 min reviewing key concepts from the past 2 weeks — quiz yourself before re-reading notes
- [ ] Run `/youtube` on at least one meaningful video watched during the week and commit the SUMMARY.md
- [ ] Check learning queue in `workspace/references/books.md` — is the current item still the right priority?
- [ ] Log deliberate practice hours in weekly priorities file (honest count, not aspirational)
- [ ] If a book was finished this week, write the distillation note within 3 days (not "later")

## Periodic

- [ ] Monthly: update `workspace/areas/learning/resources/skill-tracker.md` with self-assessed proficiency on each active skill
- [ ] Monthly: prune learning queue if it exceeds 15 items — archive items not touched in 60 days with a reason
- [ ] Monthly: check reading pace vs. goal (≥ 1 book/month) and adjust next month's schedule if behind
- [ ] Quarterly: run learning retro — what did I actually learn, what stuck, what was wasted, what is the next quarter's curriculum?
- [ ] Quarterly: align learning priorities with career goals — is what I'm learning accelerating where I want to go?
